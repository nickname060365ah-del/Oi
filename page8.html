    Bedrock แท้ — คำอธิบาย

    รวมคำสั่งที่เป็น Bedrock 100% ใช้งานได้จริง ไม่ใส่ Java command ใด ๆ

    /give @s diamond 1
/setblock ~ ~-1 ~ dirt
# ตัวอย่างคำสั่ง Bedrock (ก็อปไปใช้ได้เลย)

    ใช้งานจริง — ตัวอย่าง

    วิธีใช้ตัวอย่างคำสั่ง หรือระบบที่วงในแมพ

    1. Copy คำสั่ง
    2. วางลงใน command block
    3. ทดสอบในแมพ Bedrock
